This is tutorial slides for week7 ​ In this week, we have discussed about MACA protocol, IP address (mainly about prefix and subnetmask) and fragmentation. ​ MACA, which is an wonderful protocol to help us aviod collisions withour carrier sense medium. The basic idea is that for the sender to stimulate the receiver into outputting a short frame, so stations nearby can detect this transmission and avoid transmitting for the duration of the upcoming (large) data frame.

Tutorial_week4

This is tutorial slides for week4 ​ The first thing that we discussed in this week is two important theorems, Nyquist’s and Shannon’s theorem, in physical layer. The main difference between them is Nyquist’s theorem assums the channel that we used to transform data is noise-less while the Shannon’s theorem is only used for a noisy channel. They both have two components in their formula, bandwidth and levels of signal.
