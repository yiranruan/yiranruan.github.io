​ In this week, we have talked about deadlock, 3-way handshake and 2-army problems. Due to no solution for 2-army problems as for now, both 2-way handshake and 3-way handshake have risk to encouneter deadlock. To solve this problem, we use connection release as a example, to show you how important timer is in our system. Timer would set a time and expected to receive reply during this period. If not, the end would send their message agian.

​ Next topic is about TCP header. TCP is a popular protocol that we use for nowadays. We use port number in TCP header to identify a specific connection built between two processes. Process created by application would bind themselve with some port number if they need to use network.
